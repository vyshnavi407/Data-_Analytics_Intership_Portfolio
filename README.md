# Data-_Analytics_Intership_Portfolio
This project analyzes the Sample Superstore dataset using Python, SQL, and Pandas. It includes data cleaning, exploratory data analysis (EDA), SQL queries, and visualizations to generate insights on sales, profit, categories, and regions, presented through a structured dashboard.
# 📊 Data Analytics Internship Portfolio

## 🚀 Overview

Welcome to my Data Analytics Internship Portfolio!

This repository showcases the projects and skills I developed during a 60-Day Data Analytics Internship. The portfolio demonstrates a complete end-to-end analytics workflow, from data cleaning and exploration to dashboard creation, statistical validation, and business storytelling.

Throughout this internship, I worked with real-world datasets to extract insights, answer business questions, and create data-driven recommendations using Python, SQL, Jupyter Notebook, and GitHub.
