📊 Task 2 - Sample Superstore Data Analysis
📌 Project Overview

This project analyzes the Sample Superstore dataset using Python, SQL, and Pandas to extract meaningful business insights. It includes data cleaning, exploratory data analysis (EDA), SQL queries, and visual dashboards.

🛠️ Tools & Technologies
Python (Pandas, Matplotlib, Seaborn)
SQL (SQLite)
Jupyter Notebook
HTML (Dashboard output)

📂 Project Structure
project_folder/
│
├── dataset/              # Raw dataset (Sample Superstore)
├── sql_queries/          # SQL query files
├── charts/               # Generated visualizations
├── dashboard/            # Final dashboard (HTML)
├── task2_analysis.py     # Main analysis script
├── eda_superstore.ipynb  # Jupyter notebook
└── superstore.db         # SQLite database

📊 Key Features
Data cleaning and preprocessing
SQL-based business queries
Sales & profit analysis
Category and region-wise insights
Visual charts and graphs
Interactive dashboard creation

🚀 How to Run
Load dataset into Jupyter Notebook
Run eda_superstore.ipynb
Execute SQL queries in SQLite
Open dashboard/dashboard.html to view results

📈 Insights Generated
Most profitable product categories
Regional sales performance
Discount impact on profit
Top-performing segments
